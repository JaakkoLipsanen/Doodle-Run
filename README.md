## Doodle Run

[**Windows Phone Store**](https://www.microsoft.com/en-us/store/p/doodle-run/9wzdncrfhv2f)


Doodle Run is a simple endless runnner game that I developed for Windows Phone in late 2012. Somehow, it got really popular and it has over **1,000,000** (a million!!) downloads! There is a global leaderboard (daily/weekly/all time).
